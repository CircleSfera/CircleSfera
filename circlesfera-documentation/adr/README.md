# Architecture Decision Records (ADRs)

ADRs are the **next documentation milestone** for CircleSfera.

This folder is the planned home for Architecture Decision Records referenced by `AGENTS.md` (priority tier 4 in the source-of-truth order). Individual ADR files are not authored yet.

Until ADRs exist here:

1. Prefer `schema.prisma` and implemented Nest/React code over narrative docs.
2. Treat Abr 2026 snapshots in `circlesfera-documentation/` as historical context that may be partially stale.
3. When a durable architectural choice is made, add an ADR in this directory and link it from the relevant docs.
