# Fix Whitelist Signup Connectivity

The "failed to fetch" error is caused by a port mismatch (3001 vs 3000) and CORS block (8081 not allowed). CSRF protection would also block the request if not handled.

## User Review Required

> [!IMPORTANT]
> I am excluding the `/whitelist/signup` endpoint from CSRF protection to allow the static landing page to submit the form without needing a session-tied CSRF token. Since this is a public signup list, this is a standard trade-off for usability.

- Add `/whitelist/signup` to `excludedPaths` in the CSRF middleware.
