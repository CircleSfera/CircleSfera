### Wave 2 — Accessibility (a11y)

> [!IMPORTANT]
> 160 buttons missing `type=` and 10+ images missing `alt=` are WCAG violations.

#### All component & page files

- Add `type="button"` to every non-submit `<button>` (prevents accidental form submits)
- Add descriptive `alt` attributes to all `<img>` tags
- Add `aria-label` to icon-only buttons (e.g. like, comment, share, close)
- Add `role="dialog"` and `aria-modal="true"` to all modals

**Files affected** (highest priority):

- `StoryViewer.tsx`, `PostHeader.tsx`, `PostMedia.tsx`, `SharedPost.tsx`
- `ChatWindow.tsx`, `PhotoEditor.tsx`, `Sidebar.tsx`
- All modal components
