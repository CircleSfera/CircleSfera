## Goal Description

Refactor the frontend codebase to improve maintainability, verifyability, and performance.
Key targets:

1.  **Modularize API Services**: Split the monolithic `services/index.ts` (~400 lines) into domain-specific files.
2.  **Refactor CreatePostModal**: Decompose the complex `CreatePostModal.tsx` (~600 lines) into smaller, testable step components and a custom hook.

## User Review Required

- **Breaking Changes**: Imports from `services/index.ts` will need to be updated across the app. I will try to maintain backward compatibility via re-exports in `index.ts` initially, but shifting to direct imports is better for tree-shaking.
- **Component Logic**: The `CreatePostModal` refactor will strictly preserve existing functionality (Post/Story/Frame modes) but change the internal structure.
