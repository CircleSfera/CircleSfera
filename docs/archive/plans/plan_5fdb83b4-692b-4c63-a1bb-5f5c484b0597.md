# Implementation Plan - Follow Suggestions

## Goal Description

Implement "Follow Suggestions" to help users discover new accounts. Suggestions will be based on mutual connections (friends of friends) and popular accounts.

## Proposed Changes

### Backend

#### [NEW] [users.controller.ts](file:///Users/ShadyFeliu/Desktop/Projects/CircleSfera/backend-api/src/users/users.controller.ts)

- Add `GET /suggestions` endpoint.

#### [NEW] [users.service.ts](file:///Users/ShadyFeliu/Desktop/Projects/CircleSfera/backend-api/src/users/users.service.ts)

- Add `getSuggestions(userId)` method.
- Logic:
  1. Fetch IDs of users currently followed.
  2. Find users _not_ in that list.
  3. Prioritize by:
     - Mutual followers (Users followed by people I follow).
     - Global popularity (Total follower count).
  4. Limit results (e.g., 20 users).
