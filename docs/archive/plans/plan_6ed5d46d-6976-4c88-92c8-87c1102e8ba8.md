# Implement Chat Deletion

## Goal Description

Allow users to delete conversations. This is a new feature requested by the user ("se elimina si el usuario quiere eliminarlo").

## User Review Required

> [!WARNING]
> Deleting a conversation will remove it for ALL participants if I delete the `Conversation` record.
> **Decision**: Should it delete for everyone or just the user?
> **Assumption**: Usually "delete chat" in social apps just hides it or removes the participant's history. But for simplicity and assuming "Groups" or "Shared Conversations", if I delete the _Conversation_ entity, it's gone for everyone.
> **Refined Approach**: I will implement `deleteConversation` which checks:
>
> - If 1-on-1: Maybe just clear history for that user? (Too complex for now).
> - **Simplest implementation**: Delete the `Conversation` record. This deletes it for everyone.
> - **BETTER approach for MVP**: Since the prompt is "it is deleted if the user wants", I will implement a hard delete for now, but I'll add a confirmation dialog.
