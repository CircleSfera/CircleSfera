# Admin Audio Management

Add a "Música" tab to the Admin Dashboard so admins can manage the audio library (add/delete tracks). This enables the MusicPicker in story/post creation to offer real audio tracks.

6. Open Create Story → "Add Music" and verify the track appears
7. Delete the track from admin dashboard
8. Verify it disappears from both admin and MusicPicker
