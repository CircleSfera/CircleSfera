## Verification Plan

### Automated Tests

- Run `npm run test:e2e` in backend to verify new API endpoints (after adding them).
- Run `npm run test` in frontend if available.

### Manual Verification

1.  **Highlights**:
    - Create a highlight from existing stories.
    - Verify it appears on the profile.
    - Click to view the highlight reel.
2.  **Profile Tabs**:
    - Switch between Posts, Frames, and Saved tabs.
    - Verify content loads correctly for each.
3.  **Chat**:
    - Send a reply to a specific message. Verify unique styling.
    - React to a message with an emoji. Verify it appears for both users in real-time.
