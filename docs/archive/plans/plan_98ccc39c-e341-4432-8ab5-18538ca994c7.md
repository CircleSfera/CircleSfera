# Core Polish: Advanced Chat & Comments

This document outlines the implementation plan for the new core features requested by the user: Advanced Messaging (Groups, Typing, Receipts) and Advanced Comments (Nested Replies, Likes).

## Proposed Changes

### Feature 1: Advanced Messaging (Chat)
---
The backend `ChatService` and `AppGateway` already partially support Groups, Typing Indicators, and Read Receipts. We need to expose and connect them in the frontend UI.

#### [NEW] `CreateGroupModal.tsx`
- Build a new modal component for the Inbox to select multiple followers and create a group chat via the existing `POST /conversations` endpoint.

#### [MODIFY] [ChatWindow.tsx](file:///Users/ShadyFeliu/Desktop/Projects/CircleSfera/circlesfera-frontend/src/components/chat/ChatWindow.tsx)
- Integrate socket listeners for `user_typing` and `user_stopped_typing`. Display a dynamic "User is typing..." indicator at the bottom.
- Handle `messages_read` events to update local state mapping for Read Receipts.
- Add debounced `socket.emit('typing_start')` on text input change.

#### [MODIFY] [MessageBubble.tsx](file:///Users/ShadyFeliu/Desktop/Projects/CircleSfera/circlesfera-frontend/src/components/chat/MessageBubble.tsx)
- Render the Read Receipt indicator (✓ for sent, ✓✓ for delivered/read) based on the recipient's `lastReadAt`.

### Feature 2: Advanced Comments
---

#### [MODIFY] [schema.prisma](file:///Users/ShadyFeliu/Desktop/Projects/CircleSfera/circlesfera-backend/prisma/schema.prisma)
- Add a new `CommentLike` model with `userId` and `commentId` to allow users to like specific comments.
- Run `npx prisma db push` to synchronize the schema.

#### [MODIFY] [comments.controller.ts](file:///Users/ShadyFeliu/Desktop/Projects/CircleSfera/circlesfera-backend/src/comments/comments.controller.ts) & `comments.service.ts`

#### [MODIFY] [PostModal.tsx / CommentItem.tsx] (Frontend)

- Add the `Heart` toggle for Comment Likes next to the "Reply" button.

## Verification Plan

- Test creating a group chat with >= 2 other users.
- Test typing in a 1-on-1 chat and verify the status indicator appears remotely.
- Test replying to a comment and ensuring it indents below the parent comment.
- Test liking a comment and verifying the database updates correctly.
