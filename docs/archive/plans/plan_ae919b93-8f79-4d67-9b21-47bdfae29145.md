### 1.2 Migrate Token Storage from localStorage to HTTP-only Cookies

> [!IMPORTANT]
> This is the biggest security change. Tokens will move from `localStorage` (XSS-vulnerable) to `httpOnly` + `Secure` + `SameSite=Strict` cookies set by the backend.

#### Backend Changes

#### [MODIFY] [auth.service.ts](file:///Users/ShadyFeliu/Desktop/Projects/CircleSfera/circlesfera-backend/src/auth/auth.service.ts)

- `generateTokens()` returns tokens as before (no change to the method itself)

#### [MODIFY] [auth.controller.ts](file:///Users/ShadyFeliu/Desktop/Projects/CircleSfera/circlesfera-backend/src/auth/auth.controller.ts)

- On `login`, `register`, `refresh` responses: set `accessToken` and `refreshToken` as HTTP-only cookies via `response.cookie()`
- Cookie config: `httpOnly: true`, `secure: process.env.NODE_ENV === 'production'`, `sameSite: 'strict'`, `path: '/'`
- Access token cookie: `maxAge: 15 * 60 * 1000` (15m)
- Refresh token cookie: `maxAge: 7 * 24 * 60 * 60 * 1000` (7d)
- On `logout`: clear both cookies

#### [MODIFY] [jwt.strategy.ts](file:///Users/ShadyFeliu/Desktop/Projects/CircleSfera/circlesfera-backend/src/auth/strategies/jwt.strategy.ts)

- Extract token from cookies instead of/in addition to Authorization header
- Use `ExtractJwt.fromExtractors([cookieExtractor, headerExtractor])` for backwards compatibility

#### [NEW] [cookie.config.ts](file:///Users/ShadyFeliu/Desktop/Projects/CircleSfera/circlesfera-backend/src/common/config/cookie.config.ts)

- Centralized cookie options factory

#### [MODIFY] [main.ts](file:///Users/ShadyFeliu/Desktop/Projects/CircleSfera/circlesfera-backend/src/main.ts)

- Add `cookie-parser` middleware: `app.use(cookieParser())`
- Update CORS: `credentials: true`

#### Frontend Changes

#### [MODIFY] [api.ts](file:///Users/ShadyFeliu/Desktop/Projects/CircleSfera/circlesfera-frontend/src/services/api.ts)

- Add `withCredentials: true` to Axios instance
- Remove Authorization header interceptor (cookies are sent automatically)
- Remove manual token refresh interceptor logic (backend handles cookies)

#### [MODIFY] [authStore.ts](file:///Users/ShadyFeliu/Desktop/Projects/CircleSfera/circlesfera-frontend/src/stores/authStore.ts)

- Remove `accessToken` and `refreshToken` from persisted state
- Keep only `profile` in localStorage (non-sensitive)
- Add `isAuthenticated` flag set via API call check

#### [MODIFY] [socketStore.ts](file:///Users/ShadyFeliu/Desktop/Projects/CircleSfera/circlesfera-frontend/src/stores/socketStore.ts)

- Remove `auth.token` and `query.token` from socket config
