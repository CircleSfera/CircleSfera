## Execution Steps

### 1. Local Rebuild (Development)
- Stop all running containers.
- Rebuild the `circlesfera-shared` package.
- Rebuild and start all service containers using `docker-compose.yml`.

### 2. Production Deployment (VPS)
- Commit and push all recent changes (including verification badge fixes) to the `main` branch.
- This will trigger the GitHub Action `Deploy to OVH VPS`.
- Monitor the deployment progress.

## Verification Plan

### Automated Tests
- Run `npm test` or equivalent in critical modules.
- Check container health status after deployment.

### Manual Verification
- Access the VPS URL to verify the new verification badges are visible.
- Check the local development site to confirm the rebuild was successful.
