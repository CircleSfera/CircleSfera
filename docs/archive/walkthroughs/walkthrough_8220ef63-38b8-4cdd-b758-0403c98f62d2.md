# Walkthrough: Avatar y Username Edit en CircleSfera

## Resumen de Cambios

Se implementó funcionalidad completa para cambiar avatar y username con verificación de disponibilidad en tiempo real.

---

## Backend

### Nuevo Módulo: Uploads

Creado módulo para subir archivos:

- [uploads.module.ts](file:///Users/ShadyFeliu/Desktop/Projects/CircleSfera/backend-api/src/uploads/uploads.module.ts)
- [uploads.controller.ts](file:///Users/ShadyFeliu/Desktop/Projects/CircleSfera/backend-api/src/uploads/uploads.controller.ts)
- [uploads.service.ts](file:///Users/ShadyFeliu/Desktop/Projects/CircleSfera/backend-api/src/uploads/uploads.service.ts)

**Características:**

- Soporta imágenes (jpg, png, gif, webp) y videos (mp4, mov)
- Límite de 10MB por archivo
- Genera UUID único para cada archivo
- Sirve archivos estáticos desde `/uploads/`

### Endpoint Check Username

**GET** `/api/v1/profiles/check-username/:username`

Respuesta:

```json
{
  "available": true,
  "message": "Username is available"
}
```

## Frontend: Settings Page

### Avatar Upload

- Click en foto o "Change profile photo"
- Preview inmediato mientras sube
- Spinner de carga durante upload
- Actualiza avatar en authStore

### Username Check

- Verificación con **debounce** (500ms)
- ✅ Checkmark verde = disponible
- ❌ X roja = ocupado o inválido
- Borde del input cambia de color
- Mensaje de estado bajo el campo
